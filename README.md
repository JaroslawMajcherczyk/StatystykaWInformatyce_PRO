# Statystyka w Informatyce – Projekt (React + Vite)

Aplikacja webowa stworzona w **React + Vite** na potrzeby przedmiotu „Statystyka w informatyce”.

Repozytorium:  
`https://github.com/JaroslawMajcherczyk/StatystykaWInformatyce_PRO.git`

---

## Szybki start

Aby pobrać i uruchomić projekt:

```bash
git clone https://github.com/JaroslawMajcherczyk/StatystykaWInformatyce_PRO.git
cd StatystykaWInformatyce_PRO
npm install
npm run dev
```

Po uruchomieniu aplikacja będzie dostępna pod adresem:
http://localhost:5173
